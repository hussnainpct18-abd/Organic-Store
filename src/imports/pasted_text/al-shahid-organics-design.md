Project Name:
AL Shahid Organics – Premium Organic Products Website

Overview

Design a complete responsive website for AL Shahid Organics, a premium organic food brand that sells products such as:

Pure Organic Honey
Desi Ghee
Organic Oils
Dry Fruits
Herbal Products
Natural Foods

The website should have a premium organic aesthetic with smooth animations, elegant typography, modern UI, soft shadows, rounded corners, glassmorphism where appropriate, and beautiful product cards.

The design should feel trustworthy, luxurious, and clean while encouraging visitors to purchase organic products.

The design should include both:

Customer Website
Admin Panel

The backend will NOT use any database.

Products, images, and admin credentials will be stored in files on the server.

Design the UI while keeping this architecture in mind.
